## 📌 Internship Task

This project is part of the **Data Science Internship – February 2026**.
The objective is to build a **robust and reusable NLP preprocessing pipeline** capable of handling real-world noisy text data.

## 🚀 Project Overview

In real-world NLP applications, raw text often contains:

* Noise (emojis, symbols, repeated characters)
* Inconsistent formatting
* Unstructured patterns (URLs, numbers, slang)

This project focuses on transforming such messy text into **clean, meaningful tokens** suitable for machine learning models.

## 🎯 Key Features

* ✅ Lowercasing text for normalization
* ✅ Removal of URLs and email patterns
* ✅ Removal of numbers and punctuation
* ✅ Handling repeated characters (e.g., "soooo" → "so")
* ✅ Removal of extra spaces
* ✅ Filtering short tokens (except meaningful words like "no", "not")
* ✅ Tokenization of cleaned text
